# Denden-TV

Denden-TV est une plateforme de streaming IPTV moderne et performante, développée avec React, Vite et une architecture full-stack.

## Fonctionnalités

- **Interface moderne et intuitive** : Design épuré inspiré des meilleures applications de streaming actuelles, harmonisé avec une palette de couleurs "Sky Blue".
- **Multi-sources** : Gestion de flux IPTV (M3U8).
- **Lecteur vidéo avancé** : Lecture fluide des flux.
- **Gestion de favoris et catégories** : Navigation facile dans vos chaînes préférées.
- **Support Live & VOD** : Adapté pour le contenu en direct et à la demande.

## Technologies utilisées

- **Frontend** : React 18+, Vite, Tailwind CSS.
- **Backend** : Node.js (Express), TypeScript.
- **Déploiement** : GitHub Pages (via GitHub Actions).

## Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/mansstyle59/Denden-TV-.git
   cd Denden-TV-
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez en mode développement :
   ```bash
   npm run dev
   ```

## Déploiement

Le projet est configuré pour un déploiement automatique sur GitHub Pages grâce aux GitHub Actions.
